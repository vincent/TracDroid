h1. TracDroid
is a simple Trac browser for android.

h2. Current objectives
 * view and edit wiki pages
 * create ticket, view and edit complete ticket infos (details, changelog, actions)
 * create, view + tickets summary, edit milestones
 * attachments + screenshot tool
 * search the trac instance

h2. Requires
 * "XmlRpcPlugin":http://trac-hacks.org/wiki/XmlRpcPlugin to be installed on trac instance
